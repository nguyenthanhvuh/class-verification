CSCE990 Software Verification - READINGS

Reading List

Software Verification Overview & Model Checking

Topics: Hoare logic, predicate transformer, model checking, CEGAR, predicate abstraction

  1. Hoare, Axiomatic Basis for Computer Programming, CACM, 1969

  2. Dijkstra, Guarded commands, nondeterminacy and formal derivation of program. CACM, 1975

    Note: you might want to start by reading these 2 articles from Wikipedia: Hoare logic and Predicate Transformer

  3. JHala et al., Software Model Checking, ACM Computing Surveys, 2009 Chapter 1
  4. JHala et al., Software Model Checking, ACM Computing Surveys, 2009 Chapters 2,3
  5. Visser et al., Model Checking Programs, ASE 2003
  6. JHala et al., Software Model Checking, ACM Computing Surveys, 2009 Chapters 4,5
  7. Ball et al., Automatic predicate abstraction of C programs PLDI, 2001

Symbolic Execution & Test Case Generation

  1. King, Symbolic Execution and Program Testing, CACM 1976
  2. Sarfraz Khurshid, Corina S. Pasareanu, and Willem Visser, Generalized Symbolic Execution for Model Checking and Testing, TACAS 2003
  3. Edward J. Schwartz, Thanassis Avgerinos, David Brumley, All You Ever Wanted to Know About Dynamic Taint Analysis and Forward Symbolic Execution, Oakland 2010

Optional

  • T. Ball and S. K. Rajamani. The SLAM project: Debugging system software via static analysis. POPL, 2002

Automatic Invariants Generation

Topics: Dynamic analysis, Daikon, DIG, CEGAR

  1. Michael D. Ernst, Jake Cockrell, William G. Griswold, David Notkin, Dynamically Discovering Likely Program Invariants to Support Program Evolution, IEEE Trans. Software Eng 2001.

    Tool: Daikon

  2. Corina S. Pasareanu and Willem Visser: Veriﬁcation of Java Programs using Symbolic Execution and Invariant Generation, SPIN 2004

Optional:

  • Findler et al, Contracts for Higher-Order Functions, ICFP 2002
  • Flanagan, Cormac, and K. Rustan M. Leino, Houdini, an annotation assistant for ESC/Java, International Symposium of Formal Methods (FM), 2001.

  • Nguyen et al., Counterexample-guided Approach to Finding Numerical Invariants, FSE 2017

    Tool: DIG

Program Synthesis/Repair

  1. Alur, et al., Syntax-guided synthesis, Dependable Software Systems Engineering, 2015.

  2. Armando Solar-Lezama, Liviu Tancau, Rastislav Bodik, Vijay Saraswat, Sanjit A. Seshia, Combinatorial Sketching for Finite Programs, ASPLOS 2006.

    Tool: Sketch

  3. Westley Weimer, ThanhVu Nguyen, Claire Le Goues, Stephanie Forrest, Automatically finding patches using genetic programming, International Conference on Software Engineering, 2009

    Tool: GenProg

  4. Xuan-Bach D. Le et al., S3: Syntax- and Semantic-Guided Repair Synthesis via Programming by Examples, Foundation of Software Engineering 2017

Automated Debugging

  3. Flanagan et al., Extended Static Checking for Java, PLDI , 2002 Ten-Year Most Influential Paper Award
  4. David Hovemeyer and William Pugh, Finding bugs is easy, OOPSLA, 2004
